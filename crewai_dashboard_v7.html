⚙️ v7 God Mode Setup — Do This Once
1

Install Required Libraries

Run in VS Code terminal:

pip install python-dotenv google-ads litellm crewai langchain-openai beautifulsoup4 requests uvicorn fastapi
2

Create your .env file (Security — Never Hardcode Keys)

In the same folder as google_ads_crew_v7.py, create a file called .env and add:

# ── Required ────────────────────────────────── GROQ_API_KEY=your_groq_api_key_here # ── Google Ads (add when you have Standard Access) ── GOOGLE_ADS_DEVELOPER_TOKEN=your_developer_token GOOGLE_ADS_CLIENT_ID=your_client_id.apps.googleusercontent.com GOOGLE_ADS_CLIENT_SECRET=your_client_secret GOOGLE_ADS_REFRESH_TOKEN=your_refresh_token GOOGLE_ADS_MCC_ID=your_mcc_customer_id
3

Get Groq API Key (Free)

Go to console.groq.com → Create account → API Keys → Create Key → paste into .env

4

Get Google Ads Developer Token

Google Ads → Admin → API Center → Copy Developer Token. Apply for Standard Access.

5

Run the Server

python google_ads_crew_v7.py

Open this file (crewai_dashboard_v7.html) in Chrome.
